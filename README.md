# simple-java-database-gui

A basic JavaFX app to interact with a cryptocurrency database.

## Features

- Buttons to run preset SQL commands  
- Text field to enter custom SQL  
- Built with Maven

## Requirements

- Java 23  
- Maven  
- MySQL
- mysql-connector-java
